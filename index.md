# Test Agent – Conditional Access Viewer

## Overview

This application is a test agent built for internal evaluation purposes.
It allows authorized users to view Conditional Access Policies from a Microsoft Entra ID tenant.

## Intended Use

This application is intended for development and testing only.
It must not be used in production environments or with sensitive data.

## Data Access

The application may access:

* Conditional Access Policies
* Tenant configuration metadata

No guarantees are made regarding accuracy, completeness, or availability.

## Privacy

This application does not intentionally store personal data.
Any data retrieved is processed temporarily for display and testing purposes only.

Users should not input or expose sensitive or personal information while using this application.
